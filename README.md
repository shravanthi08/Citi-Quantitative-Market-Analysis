# Citi-Quantitative-Market-Analysis

**Executive Summary** :

This project simulates a real banking scenario where quantative methods are applied to financial market data. It demonstrates the ability to evaluate risk exposures, identify key drivers of performance, and provide strategic recommendations that support decision-making in financial services.

**Problem Statement** :

The challenge was to analyze market datasets to quantify risk, assess financial exposure, and identify variables influencing investment outcomes. The goal was to deliver insights that help Citi manage risk and optimize market strategies.

**Approach** :

**Tools Used** : Excel (data cleaning, pivot tables, regression), Word (task reports), PowerPoint(Business Presentation).

**Workflow** :

* Task 1: Mathematical foundations and scenario understanding.

* Task 2: Data cleaning and exploratory analysis.

* Task 3: Regression modeling and risk quantification.

* Task 4: Strategic recommendations and consulting deliverables.

* Combined statistical analysis with business communication (reports + Quizzes + Scenario documentation)

**Results/Impact** :

* Quantified market risk exposure using regression and statistical methods.

* Identified key financial variables influencing outcomes.

* Produced dashboards and charts to vsualize market trends.

* Delivered consulting-style reports and presentations with actionable recommendations for risk mitigation.

**Career Impact**

This project highlights the ability to apply quantitative analysis to financial risk assessment, combining technical rigor with domain expertise. It demonstrates readiness for analytics role in banking and finance where data-driven insights directly support strategic decisions.
